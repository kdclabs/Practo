Looking for appointment booking widgets? Practo Instant Appointment Booking Widget for your website.

== Description ==
With the New Practo.com Widget, your clients can now book __Instant Confirmed Appointments__ with you directly from your website!

== Installation ==
1. Unzip and upload contents of the plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Make sure you have your Practo ID ready.
